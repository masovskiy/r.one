О проекте

Ramix — это открытый хаб для просмотра и поиска аниме. Проект был создан с целью объединить любителей аниме и предоставить удобный интерфейс для поиска любимых аниме-сериалов. Сайт позволяет вам быстро найти интересующее аниме по названию или жанру.

О разработчиках

Этот проект был разработан с целью практики веб-программирования и создания полезного ресурса для аниме-сообщества. Ramix был создан благодаря совместным усилиям двух энтузиастов:

  • Роман Масовский (unimas.maiers) — разработчик фронтенда и создатель проекта.
  • Артём Рамоф (ram_off47) — милый и добрый любитель аниме, соавтор проекта.

Мы использовали современный стек технологий, включая HTML, CSS и JavaScript, а также API для работы с аниме-базами данных. Проект разрабатывался с открытым исходным кодом и доступен на GitHub.
